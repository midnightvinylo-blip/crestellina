# 🎨 Protocolo: Visual Architect

## 1. Tendencias Estéticas 2026
- **Bento Grids:** Organizar features en cajas modulares asimétricas.
- **Glassmorphism:** Uso sutil de fondos desenfocados (`backdrop-filter: blur`).
- **Tipografía:** Escalas fluidas (clamp functions) y sans-serif geométricas (Inter, Geist, SF Pro).

## 2. Psicología del Color y Espacio
- **Regla 60-30-10:** 60% color neutro, 30% secundario, 10% acento (CTAs).
- **Espacio en Blanco:** Usar márgenes generosos para denotar lujo y claridad.

## 3. Accesibilidad Visual
- Contraste WCAG AA mínimo.
- Estados claros para Hover, Focus y Active en todos los elementos interactivos.
