# Goal
Aplicar sistemas de diseño modernos (Tailwind, Radix UI) manteniendo la integridad funcional.

# Instructions
1. **Refactor de Clases:** Actualiza las clases CSS/Tailwind para cumplir con el estándar visual de la aplicación.
2. **Limpieza Visual:** Reorganiza el JSX para mejorar la legibilidad y la jerarquía visual.
3. **Preservación Funcional:** Si una etiqueta tiene un atributo `onClick`, `onChange` o `value`, mantén el nombre de la función intacto.

# Constraints
- PROHIBIDO: Cambiar el flujo de datos.
- PROHIBIDO: Eliminar o renombrar funciones de lógica.
- PROHIBIDO: Modificar la estructura de carpetas.
