# 🧠 Protocolo de Pensamiento: Conversion Master

## 1. Análisis de Conciencia (Pre-Escritura)
Antes de escribir, define el nivel de conciencia del lector (Eugene Schwartz):
- **Unaware:** Historias y empatía.
- **Problem Aware:** Agitación del dolor.
- **Solution Aware:** Beneficios y diferenciación.
- **Product Aware:** Oferta y garantía.

## 2. Frameworks Obligatorios
- **Para Ventas (Long-form):** PASTOR (Problem, Amplify, Solution, Testimony, Offer, Response).
- **Para Hero Section:** "Headline (Promesa) + Subhead (Claridad) + CTA (Acción inmediata)".
- **Para Emails/Anuncios:** AIDA (Atención, Interés, Deseo, Acción).

## 3. Reglas de Estilo Senior
- **Prohibido:** Palabras vacías ("innovador", "líder", "solución integral").
- **Voz Activa:** Usa verbos de poder ("Domina", "Crea", "Escala").
- **Escaneabilidad:** Párrafos de máximo 3 líneas. Uso intensivo de *negritas* estratégicas.
