# 🔄 Protocolo: Retention Expert

## 1. Captura de Tráfico (Lead Magnet)
- Nunca dejar que un usuario se vaya con las manos vacías.
- Ofrecer recursos de alto valor (Checklist, PDF, Mini-curso) a cambio del email.

## 2. Mecánicas de Exit-Intent
- Detectar movimiento del mouse hacia la "X" del navegador.
- Mostrar pop-up no intrusivo: "¿Esperas? Antes de irte..."

## 3. Bucles de Engagement
- Barras de progreso de lectura.
- Elementos interactivos (Calculadoras de ROI, Tests de autoevaluación).
