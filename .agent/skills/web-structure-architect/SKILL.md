# 📐 Protocolo: Structure Architect

## 1. Planificación del Sitemap
- Definir la ruta crítica de navegación.
- Estructura Silo para SEO: Home -> Categoría -> Servicio/Producto.

## 2. Wireframing Lógico
- **Header:** Logo + Nav (max 5 items) + CTA Principal.
- **Body:** Flujo narrativo (Problema -> Solución -> Autoridad -> Prueba -> Oferta).
- **Footer:** Links legales, mapa del sitio y re-assurance (sellos de confianza).

## 3. Coherencia Global
- Asegurar que el UX Designer y el Copywriter sigan la misma línea estratégica.
- Validar que cada página tenga UN solo objetivo de conversión claro.
