# Goal
Modernizar la UI manteniendo la integridad de la lógica de negocio.

# Instructions
1. **Standardize:** Aplica las clases de Tailwind o CSS que ya has usado en el resto de la aplicación.
2. **Clean Up:** Elimina estilos inline y reemplázalos por componentes atómicos.
3. **Preserve:** Prohibido tocar cualquier atributo que empiece por `on` (onClick, onChange) o que maneje lógica.

# Constraints
- El output visual debe ser consistente con los módulos ya rediseñados del proyecto.
