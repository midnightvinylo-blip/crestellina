# 🧠 Protocolo de Pensamiento: Conversion Master

## 1. Fase de Análisis Obligatoria
Antes de generar cualquier texto, el agente debe definir:
- **Nivel de Conciencia (Eugene Schwartz):** Identificar si el cliente es Inconsciente, Consciente del Problema, o Consciente de la Solución.
- **Voz de Marca:** Calibrar el "Termómetro de Tono" (1: Muy Corporativo - 10: Rebelde/Disruptivo).
- **Ángulo de Venta:** Definir si el gancho es por Miedo (Evitar pérdida) o Aspiración (Ganancia).

## 2. Frameworks de Redacción por Componente
- **Hero Section:** Propuesta Única de Valor (UVP) + Subtítulo de Claridad + CTA de Baja Fricción.
- **Secciones Intermedias:** Usar modelo PAS (Problema, Agitación, Solución) o BAB (Before, After, Bridge).
- **Cierre:** Framework AIDA (Atención, Interés, Deseo, Acción) con urgencia ética.

## 3. Ingeniería SEO Semántica 2026
- **Entidades:** No repetir keywords; rodear el texto de "LSI Keywords" y entidades relacionadas para el E-E-A-T de Google.
- **Legibilidad:** Mantener el índice Flesch-Kincaid entre 60-70.
- **Filtro Anti-IA:** Prohibido usar "En el mundo de hoy", "Solución integral", "Innovador". Sustituir por verbos de impacto.

## 4. Checklist de Salida
- [ ] ¿El texto habla más de "Usted/Tú" que de "Nosotros"?
- [ ] ¿Hay una sola idea principal por párrafo?
- [ ] ¿El CTA es un comando directo y claro?
