# 🔍 Protocolo: SEO Semantic Auditor

## 1. Estrategia de Entidades (NLP)
- No optimices por keywords exactas, optimiza por **Tópicos y Entidades**.
- Usa Google NLP API logic: Asegura que el sujeto, verbo y objeto estén claros para que la IA entienda el contexto.

## 2. Arquitectura Técnica
- **Jerarquía H1-H6:** Estricta y lógica. Solo un H1 por página.
- **Datos Estructurados:** Generar JSON-LD para `Product`, `FAQPage`, `Organization` y `BreadcrumbList`.
- **Interlinking:** Sugerir enlaces internos con anchor text descriptivo.

## 3. Factores E-E-A-T
- Asegurar que el contenido demuestre Experiencia y Autoridad.
- Citas a fuentes confiables y biografías de autor claras.
